# Nombre del Proyecto

## Descripción
Este proyecto es un juego desarrollado en Godot Engine. El juego incluye varios niveles, enemigos con inteligencia artificial avanzada, y una interfaz de usuario intuitiva. Está diseñado para ser multiplataforma, funcionando en Windows, macOS, Linux, Android e iOS.

## Características
- **Niveles**: Tres niveles jugables: "Bosque", "Cueva" y "Castillo".
- **Enemigos**: Diferentes tipos de enemigos con comportamientos únicos.
- **IA Avanzada**: Enemigos con inteligencia artificial que patrullan, atacan y se defienden.
- **Interfaz de Usuario**: Menús interactivos y HUD dinámico.
- **Localización**: Soporte para múltiples idiomas, incluyendo inglés, español y francés.

## Instalación
1. Clona este repositorio:
	```bash
	git clone https://github.com/usuario/nombre-del-proyecto.git
	```
2. Abre el proyecto en Godot Engine.
3. Ejecuta el proyecto desde el editor de Godot.

## Uso
- **Movimiento del Jugador**: Usa las teclas de flecha o WASD para mover al personaje.
- **Interacción**: Presiona la tecla "E" para interactuar con objetos.
- **Menú Principal**: Accede al menú principal desde la pantalla de inicio.
